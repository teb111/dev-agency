# DevAgency - Creative Digital Agency Website

A stunning, creative agency website built with React and modern web technologies. This website showcases our team's expertise in mobile app development, web development, consultation services, and digital problem-solving.

## ✨ Features

- **Beautiful Design**: Creative and artistic design with smooth animations
- **Responsive**: Mobile-first approach ensuring perfect display on all devices
- **Modern Tech Stack**: Built with React, Vite, Framer Motion, and Lucide React
- **Performance Optimized**: Fast loading and smooth interactions
- **SEO Ready**: Optimized for search engines
- **Contact Integration**: Frontend-only contact form that opens user's email client

## 🚀 Tech Stack

- **React 19** - Latest React with modern features
- **Vite** - Fast build tool and development server
- **Framer Motion** - Smooth animations and transitions
- **Lucide React** - Beautiful icon system
- **CSS3** - Custom design system with gradients and animations

## 🛠️ Quick Start

1. **Clone the repository**

   ```bash
   git clone <your-repo-url>
   cd agency-website
   ```

2. **Install dependencies**

   ```bash
   npm install
   ```

3. **Start development server**

   ```bash
   npm run dev
   ```

4. **Build for production**
   ```bash
   npm run build
   ```

## 🚀 Deployment

This project is optimized for Netlify deployment:

1. Connect your GitHub repository to Netlify
2. Set build command: `npm run build`
3. Set publish directory: `dist`
4. Deploy!

The `netlify.toml` file is already configured for proper routing.

## 📧 Contact Form

The contact form is designed to work without any backend dependencies:

- Collects project details
- Opens user's default email client with pre-filled content
- Provides fallback instructions if email client doesn't open
- Fully responsive and accessible
